##  Day-2 Two sum

##  Problem link- https://leetcode.com/problems/two-sum/

## Approach

Created a variable l to store the length of the array nums. Then created a nested for loop with variables i and j that traverse through the vector element wise.
Used if condition to check that:
1) the indices aren't equal
2) target - element at i= element at j
if these two conditions are satisfied together then the vector with i and j indices is returned.

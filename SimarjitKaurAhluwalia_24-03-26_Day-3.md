##  Day-3 Best time to buy and sell stocks

##  Problem link- https://leetcode.com/problems/best-time-to-buy-and-sell-stock/

## Approach

Declared integers mini set to the maximum value an integer can store and maxprof is set to 0.
Using for loop, traversed through the vector prices and updated the mini value and maxprof value.
Finally returned the value of maxprof.
